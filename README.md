# Morrow — Design Agency Homepage

A modern, responsive agency landing page built with Next.js and Tailwind CSS for the Next.js Developer Internship assignment.

## Live Demo

🔗 https://nextjs-agency-home-page.vercel.app/

## GitHub Repository

🔗 https://github.com/anthonyponson/Nextjs-Agency-HomePage
---

## Overview

Morrow is a modern digital agency homepage featuring a bold hero section, service showcase, portfolio projects, and contact section. The design focuses on clean typography, responsive layouts, smooth interactions, and an engaging user experience.

---

## Features

- Responsive design for mobile, tablet, and desktop
- Modern agency-style user interface
- Hero section with animated elements
- Services section
- Portfolio showcase section
- Contact section with validation
- Light and dark mode support
- Mobile navigation with hamburger menu
- Smooth hover and transition effects
- SEO metadata configuration
- Accessibility considerations
- Optimized images and assets

---

## Tech Stack

- Next.js 16 (App Router)
- React
- JavaScript (ES6+)
- Tailwind CSS 4
- Lucide React Icons
- Next/Image

## Installation & Setup

### Clone the repository

```bash
git clone <your-repository-url>
```

### Navigate to the project

```bash
cd nextjs-agency-homepage-design
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

### Create a production build

```bash
npm run build
```

### Start production server

```bash
npm start
```
